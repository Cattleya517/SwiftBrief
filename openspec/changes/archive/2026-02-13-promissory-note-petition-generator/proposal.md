## Why

台灣民眾在持有本票且債務人拒絕付款時，需向法院聲請本票裁定准予強制執行。目前多數人需自行撰寫聲請狀或花錢委託代書，格式要求嚴格且容易出錯。SwiftBrief 提供一個繁體中文 Web App，讓使用者透過表單填寫必要資料後，即可預覽並產生符合台灣法院格式的民事本票裁定聲請狀 PDF，大幅降低聲請門檻。

## What Changes

- 建立全新繁體中文 Web App（SwiftBrief）
- 提供結構化表單供使用者填寫：
  - 當事人資料：聲請人（債權人）與相對人（債務人）的姓名、身分證字號、住址
  - 請求標的：本票面額、利息起算日、請求裁定准予強制執行
  - 事實與理由：取得本票時間、到期日、提示付款遭拒等事實描述
  - 本票細節：發票日、票號、金額、到期日（無記載則視為見票即付）
- 即時預覽聲請狀內容
- 產生符合台灣法院格式的 PDF 檔案供下載
- 目前版本為手動填寫，未來將加入圖片辨識自動填入功能

## Capabilities

### New Capabilities
- `petition-form`: 民事本票裁定聲請狀表單 — 結構化輸入介面，含當事人資料、請求標的、事實與理由、本票細節等欄位，包含台灣身分證字號格式驗證
- `petition-preview`: 聲請狀即時預覽 — 將表單資料即時渲染為符合法院格式的聲請狀預覽畫面
- `petition-pdf`: PDF 產生與下載 — 將聲請狀內容輸出為可列印、符合台灣法院規格的 PDF 檔案

## Impact

- **新增前端 Web App**：需選定前端框架（如 Next.js / Nuxt）建立專案
- **PDF 產生**：需整合 PDF 產生套件，支援繁體中文字型
- **法律格式**：需研究並遵循台灣法院民事本票裁定聲請狀之標準格式
- **語言**：全站介面與產出文件皆為繁體中文
- **未來擴充**：架構需預留圖片辨識自動填入功能的整合空間
